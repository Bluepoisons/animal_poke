## 产品内如何使用

1. **首次启动**：展示摘要 + 链接完整《隐私政策》《用户协议》，获得明示同意后再申请相机/定位。  
2. **拒绝同意**：进入「仅浏览」模式（可看本地图鉴，不可发现/捕获/上传）。  
3. **撤回同意**：设置中提供入口，撤回后停止相关处理并引导删除/导出。  
4. **未成年人**：注册或首次游玩前完成年龄确认；未成年人适用防沉迷规则。
