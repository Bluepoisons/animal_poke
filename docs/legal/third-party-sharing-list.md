# Animal Poke 第三方信息共享 / 委托处理清单

**版本号**：v1.1.0  
**更新日期**：2026-07-10  

为保障相关功能实现，我们可能将必要信息 **委托** 下列服务商处理。我们要求其按照约定目的、方式处理，并采取安全保护措施。  
**客户端不直连下列业务 API、不内置第三方业务 Key**；均由我方 Go 后端代理调用。

更完整的技术说明见：[SDK 与第三方服务清单](./sdk-and-services.md)。  
合同主体全称 / 隐私政策 URL 请按 [待补充信息清单](./FILL-IN-CHECKLIST.md) 回填。

---

## 一、当前工程已实装（服务端委托）

| 类型 | 产品 / 接口（代码） | 合同主体（待回填） | 处理信息 | 目的 | 方式 |
|------|---------------------|-------------------|----------|------|------|
| 地图 / 逆地理 | **腾讯位置服务** `apis.map.qq.com/ws/geocoder/v1/` | 【待回填】 | 经纬度（可粗化） | 解析省 / 市 / 区 | 后端 HTTP；结果可缓存 |
| 天气 | **彩云天气** `api.caiyunapp.com/v2.5/.../daily.json` | 【待回填】 | 经纬度 | 一周天气、玩法与状态修正 | 后端 HTTP；可缓存 |
| 视觉大模型 VLM | 可配置 `VISION_ENDPOINT` + Model（OpenAI 兼容） | 【待回填】 | **图像瞬时** | 动物检测 / 分析 | 后端上传，**推理后销毁原图** |
| 文本大模型 LLM | 可配置 `LLM_ENDPOINT` + Model（OpenAI 兼容） | 【待回填】 | 结构化文本 | 属性 / 叙事生成 | 后端调用 |
| 关系型数据库 | **MySQL**（自建或云厂商托管） | 【待回填云厂商或自建】 | 设备、进度、同意、审计等 | 持久化 | 基础设施 |
| 缓存 / 限流（可选） | **Redis**（`REDIS_URL`，可选） | 【未启用则无】 | 限流计数等 | 共享限流 | 未配置则进程内实现 |

## 四、重要说明

1. **委托处理 ≠ 出售个人信息。**  
2. 涉及敏感个人信息（位置、可能含敏感内容的图像）的处理，依法取得同意；图像按产品策略 **不落盘、不入训练**。  
3. 新增 SDK 或改变处理目的 / 种类时，将更新本清单与《隐私政策》并告知。  
4. 上线前须将「【待回填】」替换为真实法律主体名称与隐私政策链接。

## 五、您如何减少共享

- 系统设置中关闭相机 / 定位；  
- 应用内撤回同意、使用仅浏览模式；  
- 卸载并申请删除服务端数据（见《隐私政策》）。
