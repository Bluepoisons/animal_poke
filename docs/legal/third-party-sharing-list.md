# Animal Poke 第三方信息共享 / 委托处理清单

**版本号**：v1.0.0  
**更新日期**：2026-07-10  

为保障相关功能实现，我们可能将必要信息 **委托** 下列类型的服务商处理。我们要求其按照约定目的、方式处理，并采取安全保护措施。  
**客户端不直连第三方、不内置第三方 Key**；均由我方 Go 后端代理调用（架构约定）。

| 类型 | 服务商（示例，上线前填写具体主体） | 处理信息 | 目的 | 方式 |
|------|--------------------------------------|----------|------|------|
| 地图 / 逆地理 | 腾讯地图等（待确认合同主体） | 经粗化或原始的经纬度 | 解析城市 / 区域 | 后端 API 调用 |
| 天气 | 彩云天气等（待确认） | 经纬度或城市 | 天气玩法与状态修正 | 后端 API 调用 |
| 视觉大模型 VLM | 【待定云厂商 / 模型服务】 | 图像二进制（瞬时） | 动物检测与分析 | 后端上传，**用后销毁** |
| 大语言模型 LLM | 【待定】 | 结构化文本 / 分析中间结果 | 生成属性描述等 | 后端调用 |
| 云主机 / 数据库 | 【待定云厂商】 | 账号进度、日志等 | 托管与存储 | 基础设施 |
| 对象存储（若启用） | 【待定】 | 用户主动分享的图片等 | 分享内容托管 | 可选功能 |
| 消息推送（若启用） | 【待定】 | 设备推送标识 | 通知 | 可选，另行同意 |
| 统计与崩溃（若启用） | 【待定】 | 设备信息、崩溃栈 | 质量分析 | 可选，可开关 |

## 重要说明

1. 上线前将在本表 **替换为真实法律主体名称、隐私政策链接与联系方式**。  
2. 委托处理不等同于「出售个人信息」。  
3. 如新增 SDK 或改变处理目的 / 种类，将更新本清单并依法告知。  
4. 涉及敏感个人信息的第三方处理，将依法取得单独同意（法律另有规定除外）。

## 您如何关闭

- 系统权限：关闭定位 / 相机可停止对应采集；  
- 应用内：撤回同意、仅浏览模式；  
- 卸载应用并申请删除服务端数据。
