设计系统

🎨 配色

主色 #FF8C42（按钮/高亮）

深橙 #E67300（阴影/标题）

奶油底 #FFF8F0

文字 #4A2C1A

🃏 卡片

圆角 16px、3px 稀有度边框、4px 实色下沿阴影。卡片内仅展示：动物照片、编号、地点、日期。不出现「猫/狗/鹅」等物种文字。

🗺️ 地图

手绘风格底图（绿/河/山区域色块），爪印/定位标记，点击标记底部弹出照片浮卡。地点通过 Go 后端调用腾讯地图 API 反查，本地存经纬度+地址。

📐 布局

图鉴 3 列网格；顶部状态栏+搜索+地图入口；底部 5 Tab（Profile/Collection/Camera/Fight/Store）。Camera 按钮外凸。

Godot 实现要点

CollectScreen

用 TextureRect 做照片卡，数据字段增加 photo_path / capture_date / location_name / lat / lng。移除 emoji 与 species name，保留 rarity 边框色。

MapScreen

TextureRect 地图底图 + Marker 自定义 Control（爪印标记+缩略图）。Marker 数据从 CollectScreen 共享。地点名称由后端 /geo/reverse 接口提供。

数据流

捕获时写入：photo_path, capture_ts, lat, lng。启动时若 location_name 为空，调后端腾讯地图逆地理编码补全地址。
