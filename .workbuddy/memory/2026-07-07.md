# 2026-07-07 工作记录

## 亲密度与区域声望系统补充

基于用户与团队成员的聊天记录（讨论好感度、金币设计、天梯赛制、宠物属性差异化等），将讨论结果结构化并写入游戏开发计划.md：

1. **5.6 亲密度与区域声望系统（新增）**：放弃"同一只动物识别"的不可行方案，改为个体亲密度（玩家与宠物互动）+ 区域声望（玩家与城市绑定）双轨设计。包含详细的亲密度等级表、区域声望等级表、亲密度道具价格表。

2. **5.2 宠物属性差异化**：在六维属性基础上增加物种基础偏向（猫高SPD/狗高HP/鹅高DEF）+ 品种特性（同物种不同品种±10%浮动）+ 个体随机（±15%浮动），形成三层差异化结构。

3. **5.3 拍摄质量评分**：从5维扩展到6维，新增"拍摄角度"维度（正面/正侧面=2分、背面/斜侧面=1分、遮挡=0分），回应聊天记录中"不同角度拍摄有不同分数"的讨论。

4. **6.5 天梯排位赛详细规则**：每日5场限制、青铜到传说6大段位、积分规则（胜+15~25/负-10~15）、保护机制（青铜不掉分/保级护盾/连胜加成）、8周赛季结构、赛季奖励表、区域声望联动。

5. **6.4 道具商店**：新增5种亲密度道具（高级零食30金、爱心玩具50金、纪念相框80金、亲密度礼物盒100金、遗忘之水500金），与感冒药/体力药剂形成多元消耗矩阵。

## 关键设计决策
- "同一只动物识别"技术上不可行（无唯一生物标识），明确放弃此路线，改用区域声望替代。
- 天梯每日5场限制+段位保护，确保"养老"节奏，减少挫败感。
- 亲密度道具作为金币新出口，避免经济系统通胀。
